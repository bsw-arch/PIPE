# PIPE Bot - Wolfi Micro-Container Structure

See pipe-wiki-bot FOLDER-STRUCTURE.md for complete details.

This bot follows the standard PIPE bot structure with:
- agents/ - Python bot code
- containers/ - Wolfi container definitions
- sbom/ - Software Bill of Materials
- build/ - Build automation
- ci/ - CI/CD pipelines
- docs/ - Documentation
- iac/ - Infrastructure as Code
- tests/ - Testing
- wiki/ - Wiki content
